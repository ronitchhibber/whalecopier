System Methodology

üìä Whale Selection

Whales are ranked using a multi-factor quality score combining:

  • Win Rate (30%): Percentage of profitable months (target: >60%)
  • Sharpe Ratio (40%): Risk-adjusted returns (target: >1.5)
  • Consistency (20%): Stable performance over time
  • Volume (10%): Total trading activity

Scores are normalized using MAD-based z-scores with James-Stein shrinkage to reduce estimation error.
